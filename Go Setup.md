Go Setup
========

Get GVM. https://github.com/moovweb/gvm

Make sure to add this to your `.zshrc`:

```sh
    # Source GVM
    export GVM_ROOT=/cygdrive/c/Users/CMCDragonkai/.gvm
    . $GVM_ROOT/scripts/gvm-default
```
